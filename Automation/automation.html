Scénarios de Test

J'ai créé les scénarios de test pour :

  • Connexion

  • Inscription

  • Ma Carte


  • Recherche

  • Blog
  • Contact
    • Cas de Tests

    Dans le projet créé dans Excel, nous trouverons les cas de test écrits pour :
    • Test_e2e -> L'utilisateur a effectué un achat de l'article souhaité, en passant par toutes les étapes.
    • Test_Inscription -> L'utilisateur a créé un compte en utilisant des données valides.
    • Test_Connexion -> L'utilisateur s'est connecté à son compte en utilisant les données spécifiées.
    • Test_Contact -> L'utilisateur a envoyé un message de contact avec les données spécifiées.
    • Test_Blog -> L'utilisateur a ajouté un commentaire au blog d'intérêt.
